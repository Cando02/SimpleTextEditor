# 简易文本编辑器 (SimpleTextEditor)

## 项目介绍

这是一个基于Qt框架开发的简易文本编辑器，提供了基本的文本编辑和格式设置功能。

## 功能特点

### 文件操作
- **新建**：创建新的空白文档
- **打开**：打开已有的文本文件
- **清空**：清空当前文档内容

### 编辑操作
- **剪切**：剪切选中的文本
- **复制**：复制选中的文本
- **粘贴**：粘贴剪贴板中的内容

### 格式设置
- **斜体**：将选中的文本设置为斜体
- **粗体**：将选中的文本设置为粗体
- **下划线**：为选中的文本添加下划线
- **字体大小**：调整文本字体大小（8-60）
- **字体选择**：选择不同的字体

### 界面元素
- **工具栏**：包含常用操作按钮
- **状态栏**：显示当前文件路径和字体大小进度条

## 技术实现

- **开发框架**：Qt
- **编程语言**：C++
- **界面设计**：Qt Designer
- **核心组件**：QMainWindow, QTextEdit, QToolBar, QStatusBar
- **信号槽机制**：处理用户交互事件

## 安装与运行

### 环境要求
- Qt 5.0 或更高版本
- C++ 编译器

### 编译步骤
1. 使用Qt Creator打开 `QtUserUiTextEdit.pro` 文件
2. 构建项目
3. 运行应用程序

## 使用方法

1. **新建文档**：点击工具栏中的"新建"按钮或选择"文件"菜单中的"新建"选项
2. **打开文件**：点击工具栏中的"打开"按钮或选择"文件"菜单中的"打开"选项
3. **编辑文本**：在文本编辑区域输入或修改文本
4. **设置格式**：
   - 选中要设置格式的文本
   - 点击工具栏中的斜体、粗体或下划线按钮
   - 使用字体大小 spinbox 调整字体大小
   - 使用字体下拉菜单选择字体
5. **编辑操作**：
   - 选中文本后，点击工具栏中的剪切、复制按钮
   - 点击粘贴按钮粘贴剪贴板内容
   - 点击清空按钮清空当前文档

## 功能实现细节

### 字体格式设置
- 斜体、粗体、下划线功能通过 `QTextCharFormat` 实现
- 字体大小和字体选择通过 `QSpinBox` 和 `QFontComboBox` 实现

### 状态栏
- 显示当前打开的文件路径
- 显示当前字体大小的进度条

### 编辑状态
- 剪切和复制按钮会根据是否有选中文本自动启用/禁用
- 粘贴按钮会根据剪贴板是否有内容自动启用/禁用
- 格式按钮会根据当前选中的文本格式状态自动更新
